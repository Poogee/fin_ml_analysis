# Оптимизация портфеля на основе машинного обучения с подкреплением

Курсовая работа, ФКН НИУ ВШЭ, 2026.

## Структура
- `src/` — исходный код (среды, агенты, признаки, данные, оценка)
- `configs/` — конфигурации экспериментов
- `notebooks/` — Jupyter ноутбуки
- `results/` — результаты экспериментов
- `report/` — отчет (Typst)
- `tests/` — тесты

## Данные
Данные (parquet, ~3 ГБ) не включены в репозиторий. Положите файлы в `data/`.

## Запуск
```bash
python -m src.rl.train_meta_v3  # обучение Meta-RL
typst compile report/main.typ   # компиляция отчета
```
